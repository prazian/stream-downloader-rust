# Disclaimer

## Purpose of this project

This repository is a **technical demonstration and learning resource**. It exists to show how media discovery and HTTP progressive download can be implemented in Rust, including:

- different extraction strategies per website (APIs, embedded JSON, HLS, innertube-style clients, and similar patterns);
- how to structure such a system for maintainability (site plugins, shared helpers, quality selection);
- practical performance techniques (parallel range downloads, token refresh, ffmpeg integration, and related topics).

It is intended for developers who want to study systems programming, networking, and software architecture—not as a turnkey tool for bulk media acquisition.
